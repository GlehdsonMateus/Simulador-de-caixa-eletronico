# 💳 Caixa Eletrônico

Um sistema bancário simples desenvolvido com **HTML, CSS e JavaScript**, simulando as principais operações de um caixa eletrônico.

## 📋 Funcionalidades

- ✅ Cadastro de contas
- ✅ Exclusão de contas
- ✅ Depósito
- ✅ Saque com validação de saldo
- ✅ Transferência entre contas
- ✅ Histórico de operações
- ✅ Atualização automática dos saldos

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## 📸 Interface

O sistema possui uma interface simples onde é possível:

- Cadastrar novas contas
- Selecionar contas cadastradas
- Realizar depósitos e saques
- Transferir valores entre contas
- Visualizar o histórico das operações
